# Next.js Project Template

Next.js の開発用テンプレート

## ディレクトリ構成

```
.
├── __tests__/
│   ├── __snapshots__/
│   │   └── snapshot.tsx.snap
│   ├── index.test.tsx
│   └── snapshot.tsx
├── public/
├── src/
│   ├── axios.ts
│   ├── components/
│   ├── pages/
│   ├── states/
│   │   ├── atoms/
│   │   └── selectors/
│   └── styles/
├── .eslintrc.json
├── .gitignore
├── .prettierrc
├── README.md
├── jest.config.js
├── jest.setup.js
├── next.config.js
├── package.json
├── tsconfig.json
└── yarn.lock
```

## ライブラリ

- Axios
- Recoil
- Jest
- React Tesing Library

## 環境構築

1. `git clone`
2. `yarn install`
3. `yarn dev`
